Este site tem como objetivo disponibilizar dados de correlação entre emissões de CO2 (Dióxido de Carbono) e o PIB (Produto Interno Bruto), de 2000 a 2020.



Os dados utilizados para a relação entre os dados são originados do World Bank (PIB) e Climate Watch (CO2), ambos sendo dados abertos. Por meio da barra de pesquisa inicial é possível buscar o país que deseja visualizar a relação, podendo também baixar os dados, para que possam ser usados por outros livremente.
